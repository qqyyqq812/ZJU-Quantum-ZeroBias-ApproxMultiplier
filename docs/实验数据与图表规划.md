# 论文 Evaluation 章节：实验数据与图表规划大纲 (最新终局定稿)

> [!IMPORTANT]
> **关于本作的学术定名升级与研究逻辑基盘**
> 
> * **最新定名**：全面杜绝工程化的 `KMap_V3_Final_SchemeD` 等冗杂拼写，本系列架构在学术出版中统一命名为 **`ZBA-R4ABM`** (Zero-Bias Approximate Radix-4 Booth Multiplier)。对于不同截断档位，可采用后缀化如 `ZBA-R4ABM-4A0E`。
> * **乘法器种类**：Radix-4 Booth 有符号高能效算力网络近似乘法器。
> * **研究逻辑驱动 (Motivation)**：传统下行截断固然能暴降 PPA，但必然诱发极度严重的符号偏移（Bias Error），致使庞大的神经网络中产生雪崩式误差积累。为了兼得鱼和熊掌（即深度残缺下的高度鲁棒性），我们打破树状依赖，创新启用了**“底层扁平网络重拼 + 零点行特赦法 (Row-0 Amnesty)”**。在暴力截断 4 行部分积后，硬是将全景的 Mean Error (ME) 引力拉回绝对的 0.0000 之中，达成全统计均值无偏。彻底颠覆了“高误差率必导致网络崩盘”的定论！

### 第一重逻辑：面积压榨与防偏补偿的演进博弈 (架构变体说明)
为了向 Reviewer 清晰揭示我们是如何逼近理论极限的，我们在实验图表中布设了四代对比阵列：
1. **`EXACT_BM`**：经典的精确 8-bit Radix-4 算子（基线控制组）。
2. **`Pure_4Row_ABM` (粗暴截断版)**：最原始的近似基石。直接手起刀落切除加法树最底部的 4 行（全数丢弃低位部分积）。虽然 PPA 前期下降了近 29%，但由于残差极性单向崩塌，会导致网络预测系统发生致命的方向漂移。
3. **`BARC_4Row_ABM` (外挂电路补偿版)**：同样基于 KMap 近似压缩演变而来。但为了修复前者的极性崩塌特设了 **B**ooth **A**pproximation **R**ecovery **C**ircuit 作为补偿。其破局难点在于采用的是一种“外部外加电路”的补丁法。因为没有做到彻底的内部资源复用，额外增加的纠错门控使得其面积红利立刻反缩倒退（仅剩 19%），得不偿失。
4. **`ZBA-R4ABM-4A` (内建复用王冠版)**：同源自 KMap 阵列，但彻底摒弃了外部打补丁的割裂模式！它直接自底向上将加法网络重排扁平化，并通过内部可用信号的**极致复用**在硬连线层面上天然内嵌了特赦逻辑（Row-0 本源修复）。这种将绝对零均值“原生态合拢”的做法，不仅在物理上达到了 $ME=0$ 的对冲死角，而且毫不损失性能，一脚把面积重新踩回出 -34.54% 的断崖红利！

### 第二重逻辑：系统级抗跌双轨试金石 (为何待测？)
在锁死了底层架构的物理红利后，我们需要向 Reviewer 表明它不是一个花瓶。图表的最后两列是**系统与大网评估级别**的验收栏，此区域需被预留为系统级的宏观测试锚点：
* **Zero-Shot 精度 (暴力换器推理)**：即不对抗跌落能力做任何包庇，直接拔出精确架构并在底层硬换上新研制的近似算子阵列。不更新神经网络的权重！这测试的是其**零偏差补偿 (Zero-Bias)** 的天生硬度是否能在卷积网络里做到极性自对抗。
* **微调精度 (1 Epoch 硬件感知快弹)**：属于轻型修复测试。即便我们的新架构有着底层偏差，只要允许深度神经网络反向试探 1 个周期（1 次反向传播）让其摸透这群新硅块的特性，它就必须具备瞬间恢复到原始基线高精度的强悍回血能力！
